# 常见传染病

多数儿童传染病由病毒引起、可自限，护理以对症、补液、隔离和观察红旗为主；部分靠疫苗预防。

## 常见传染病对照

| 疾病 | 典型特点 | 护理/预防要点 |
|---|---|---|
| 手足口病 | 手、足、口腔疱疹/溃疡，发热 | 对症、补液；口腔痛致拒食注意脱水；勤洗手隔离 |
| 疱疹性咽峡炎 | 咽部疱疹、高热、咽痛 | 同上；注意进食与补液 |
| 水痘 | 分批出现的红疹→水疱→结痂，瘙痒 | 防抓挠继发感染；疫苗预防 |
| 麻疹 | 高热、咳嗽、结膜炎、口腔麻疹黏膜斑、皮疹 | 及时就医；疫苗预防 |
| 风疹 | 低热、皮疹、耳后淋巴结肿大 | 对症；疫苗预防 |
| 猩红热 | 发热、咽炎、砂纸样皮疹、草莓舌 | 细菌性，需就医用抗生素 |
| 幼儿急疹（玫瑰疹） | 高热 3～4 天热退疹出 | 对症；热退疹出为特征 |
| 流行性腮腺炎 | 腮腺肿痛、发热 | 对症；注意后续听力；疫苗预防 |
| 中耳炎 | 耳痛、发热、烦躁、抓耳 | 就医评估是否需抗生素 |

## 通用护理与预防

- 勤洗手、餐具消毒、患病期居家隔离，避免交叉感染。
- 对症退热补液，密切观察精神与进食。
- 按程序接种疫苗是最有效的预防手段（见 [[免疫接种]]）。

## 何时就医

- 高热持续、精神差、拒食脱水、呼吸困难、皮疹伴颈强直/嗜睡（警惕脑膜炎）、抽搐。
