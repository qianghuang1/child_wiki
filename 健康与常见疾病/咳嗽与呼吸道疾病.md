# 咳嗽与呼吸道疾病

咳嗽是清除气道分泌物的保护性反射，多数由病毒性上呼吸道感染引起，可自限。关键在于识别需要就医的**呼吸窘迫**信号。

## 普通感冒（上呼吸道感染）

- 症状：流涕、鼻塞、打喷嚏、咳嗽、可有低热。
- 处理：多休息、补液、生理盐水滴鼻/清理鼻腔；对症为主。
- **不建议给幼儿自行用复方感冒药/镇咳药**（尤其低龄）。

## 常见呼吸道疾病对照

| 疾病 | 典型特点 | 就医要点 |
|---|---|---|
| 毛细支气管炎 | 多见于婴儿，喘息、呼吸费力 | 呼吸急促/费力、口唇发绀、拒奶 → 就医 |
| 肺炎 | 发热、咳嗽、呼吸急促 | 精神差、呼吸困难、持续高热 → 就医 |
| 喉炎/哮吼 | 犬吠样咳、声音嘶哑、吸气喉鸣 | 明显喉鸣、呼吸困难 → 急诊 |
| 百日咳 | 阵发性剧咳伴鸡鸣样回声 | 疑似即就医；靠疫苗预防 |
| 流感 | 高热、全身酸痛、乏力 | 高危儿/重症就医；可用抗病毒药、疫苗预防 |

## 何时就医（呼吸道红旗）

- 呼吸急促、费力，鼻翼扇动、肋间/胸骨凹陷。
- 口唇或面色发绀。
- 喘息明显、吸气喉鸣。
- 精神萎靡、拒食脱水、持续高热。
