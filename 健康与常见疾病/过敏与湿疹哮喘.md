# 过敏与湿疹哮喘

## 食物过敏

- 常见致敏食物：牛奶、鸡蛋、花生、坚果、小麦、大豆、鱼、海鲜。
- 过敏表现常按时间递进：
  1. 最早——口腔过敏（口周红肿、舌痒）。
  2. 中期——消化道（腹泻、呕吐、拒食）+ 生长缓慢。
  3. 后期——皮肤（湿疹）→ 呼吸道（咳嗽、喘息）。
- 母乳宝宝也可能通过母亲饮食中的蛋白致敏；可调整母亲饮食，继续母乳。

### 回避 + 激发试验（判断金标准）

1. 停用可疑食物 2～4 周 → 症状消失。
2. 再次尝试 → 症状复现即可判断。
3. 只做**一次**激发，不反复刺激（会加重）。停用食物一般 3 个月后可再试。

## 湿疹 vs 痱子

| 特征 | 痱子 | 湿疹 |
|---|---|---|
| 形态 | 密集红点 | 片状、脱屑 |
| 触感 | 轻微扎手 | 脱屑、裂口、渗水 |
| 出现速度 | 快（1～2 小时） | 慢（多日） |
| 季节 | 夏季多见 | 常年可有 |

- 湿疹管理：保湿为核心（勤涂润肤霜）、避免过热出汗和刺激、必要时遵医嘱用药。

## 哮喘

- 表现：反复喘息、咳嗽（尤其夜间/运动后）、呼吸困难。
- 管理：识别并回避诱因、遵医嘱用吸入药物；呼吸困难加重及时就医。

## 荨麻疹与过敏性鼻炎

- 荨麻疹：风团、瘙痒，找诱因、对症处理。
- 过敏性鼻炎：季节性/常年鼻痒、喷嚏、清涕；回避过敏原、遵医嘱治疗。

## 严重过敏（过敏反应/anaphylaxis）应急

- 表现：全身荨麻疹、口唇/面部肿胀、呼吸困难、喘鸣、呕吐、面色差、意识改变。
- **立即拨打急救**；如已确诊并配有肾上腺素笔，按医嘱使用；保持平卧、监测呼吸。
