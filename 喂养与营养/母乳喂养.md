# 母乳喂养

母乳是 0～6 个月婴儿的最佳食物，含丰富营养素、免疫活性物质和水分，能满足此阶段全部营养需求。

## 优点

- 满足 0～6 个月全部营养需求，有助大脑发育。
- 降低感冒、肺炎、腹泻等风险；减少成年后肥胖、糖尿病、心脑血管病等慢性病风险。
- 增进亲子关系。
- 帮助母亲：减少产后出血，降低乳腺癌、卵巢癌风险。

## 开奶与初乳

- 出生后**尽早皮肤接触、早吸吮、早开奶**（产房内 30 分钟内更佳）。
- **初乳**（产后前 5 天的黄色浓稠乳汁）抗体和免疫因子含量最高，第一口应喂初乳，帮助建立肠道菌群与被动免疫。
- 提倡母婴同室、按需哺乳，每日 8～10 次以上。

## 常见含乳姿势

| 姿势 | 特点 / 适用 |
|---|---|
| 摇篮式 | 最常用；宝宝头枕在妈妈臂弯，前臂托身体 |
| 交叉摇篮式 | 头托在手上，便于观察含乳，适合新手 |
| 橄榄球式 | 手臂托宝宝腿部；适合乳头扁平/凹陷、双胞胎、剖宫产 |
| 侧卧式 | 母婴同侧躺；夜间省力但**有妈妈睡着压到宝宝的窒息风险**，需清醒看护 |

## 正确含乳（含接）

- 先用乳头轻触宝宝嘴唇，诱发觅食反射，待嘴张大再送入乳房。
- **含接良好**：含住大部分乳晕（上方露出比下方多）、嘴张大、脸颊鼓圆、能听到吞咽声、吃完表情满足。
- **含接不良**：只含乳头（妈妈疼痛）、嘴唇内卷、脸颊凹陷、发出「咔哒」声。
- 纠正：用手指伸入嘴角解除吸附，重新含接。

## 如何判断宝宝吃饱

- 每天哺乳 8～12 次，每侧 15～20 分钟以上。
- 每次至少排空一侧乳房；有节律吞咽和吞咽声。
- 每天 6 次以上小便，尿色清。
- 大便颜色按时过渡（胎便→过渡便→3～4 天转金黄色）。
- 吃完表情满足。

> 宝宝哭闹**不一定是饿**：先排查尿布、温度、过度刺激、疲倦等。

## 促进泌乳

- **宝宝充分吸吮**是最有效的方法（供应随需求增加）。
- 妈妈心情愉悦、睡眠充足、营养均衡、家人支持。
- 持续不足时在医生评估指导下处理，不要轻易放弃。

## 常见障碍

| 问题 | 要点 |
|---|---|
| 乳头扁平/凹陷 | 哺乳前牵拉；严重时用乳头矫正器（每次约 3 秒间歇） |
| 乳腺炎（未成脓） | 红肿热痛+发热；继续哺乳+手挤排空；发热>38.5℃就医 |
| 乳腺脓肿 | 需就医，可能切开引流；健侧继续哺乳 |
| 乳头混淆 | 奶瓶/安抚奶嘴更省力致拒乳；先刺激喷乳反射再喂 |
| 母乳不足 | 信心+频繁哺乳+休息+营养；混喂时先母乳后配方奶 |
| 母乳过多 | 哺乳前热敷软化、每次喂一侧、喂后冷敷；勿额外排空 |

详见 [[常见喂养问题]]。

## 哺乳期用药

- 多数药物哺乳期可用（L1/L2 级乳汁转移少）；应在医生指导下使用。
- 原则：最小有效剂量、优选单一成分、外用>口服>静脉；担心时哺乳后服药。
- 中草药多缺乏安全数据，不建议自行使用。
- 可参考 LactMed 等哺乳用药数据库。

## 何时就医 / 需评估

- 出生体重下降 **>7%** 且持续或吃奶差 → 评估。
- 乳腺炎发热 >38.5℃ 或形成脓肿。
- 宝宝小便明显减少、精神差、黄疸加重。
