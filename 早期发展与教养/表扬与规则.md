# 表扬与规则

## 科学夸奖六原则

1. **具体**：指出具体行为，而非空洞的"真棒"。
2. **肯定努力**：表扬坚持和尝试，而非只夸"聪明"。
3. **真诚**：不夸大其词（夸张会让孩子"免疫"）。
4. **不贬低他人**：避免"比别的小朋友都棒"式对比。
5. **就事论事**：不翻旧账、不预测未来。
6. **适时当众肯定**：增强成就感，强化正确行为。

## 建立规则

- 在保证安全前提下给予自由探索空间，同时适时建立**合理、一致**的规则。
- 帮助孩子识别自己和他人的情绪，发展自我调节能力。
- 全家保持一致的养育态度（见 [[隔代养育沟通]]）。

## "听话"与独立

- 真正的"听话" = 倾听 + 独立思考 + 活学活用，而非机械顺从。
- 鼓励孩子表达想法、参与力所能及的选择，培养自主性（不包办代替）。

## 自律的培养

- 自理能力和良好习惯在日常生活中逐步养成。
- 建立规律作息、让孩子承担适龄责任，逐步发展自我管理能力。
