# 情绪与行为管理

## 情绪管理四步

1. **认识**：帮孩子认识自己的情绪。
2. **认同**：接纳负面情绪（生气、难过都可以有）。
3. **表达**：教会适当的表达方式。
4. **疏导**：陪伴帮助走出困境。

家长的角色更像**伙伴**（平等、尊重、一起成长），而非高高在上的指导者。

## 发脾气（tantrums）

- 幼儿因语言和自控有限，容易发脾气，属发育阶段常见现象。
- 预防：规律作息、避免过度疲劳饥饿、给予有限选择、提前预告转换。
- 发作时：保证安全、保持平静、不妥协于不合理要求，事后平静沟通。
- 频繁剧烈、伴自伤或发育倒退 → 咨询医生。

## 打人、咬人

- 多因表达受限或试探边界；用简短明确的语言制止，示范替代行为，事后引导。
- 避免以暴制暴。

## 说谎的发展阶段

| 年龄 | 特点 |
|---|---|
| 3 岁前 | 无意识（想象与现实混淆） |
| 3～5 岁 | 有意识但漏洞百出 |
| 5 岁以上 | 逐渐有策略性 |

- 预防说谎的环境：平等沟通 > 指责、信任 > 监视、理解 > 惩罚；当惩罚的代价远大于诚实承认时，孩子更倾向隐瞒。

## 挫折教育

- 正确做法：**陪孩子一起面对**困难 + 提供心理支持；而非人为制造困难让其独自承受。

## 执行功能发育

- 情绪相关脑区（杏仁核）发育较早，负责决策与自控的前额叶要到 20 岁左右才成熟——对幼儿的自控不宜有超龄期待。
