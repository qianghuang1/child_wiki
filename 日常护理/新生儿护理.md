# 新生儿护理

## 抱与放

- 始终**先托稳头颈**（抱起时最先支撑、放下时最后松开）。
- 动作轻柔，交接他人时确保头部有支撑。

## 换尿布

- 通常每约 2 小时或有便时更换；用温水和软布清洁，拍干。
- 腰部留约两指空隙；尿布上缘向下折避免摩擦、打湿脐带残端。
- 睡眠中若未便可不必频繁更换。

## 穿脱衣

- 选前开襟、无绳带的衣物；穿袖时把袖子先叠拢再引手臂穿过。
- 注意保护头部、避免蒙住引起不安。

## 脐带护理

- 残端约 2 周脱落，颜色由黄→棕→黑。
- 每天 1～2 次用 **75% 医用酒精**棉签清洁根部并擦干；不用紫药水/碘伏（会遮盖分泌物观察）。
- 尿布上缘下折防止打湿摩擦；脱落后继续清洁 3～7 天至愈合。
- 红旗：大量渗液、脓性分泌、异味、周围发红 → 就医。

## 常见特殊生理现象（多为正常，会自行消退）

| 现象 | 说明 |
|---|---|
| 红色尿 | 尿酸盐浓度高；多饮后约 72 小时消退，持续需评估 |
| 马牙 | 上皮堆积/黏液，勿挑破，数月自消 |
| 外耳形状异常 | 宫内受压所致，数周自行矫正 |
| 乳房肿胀（男女均可） | 母体激素影响，数周消退，勿挤压 |
| 眼分泌物/泪囊阻塞 | 鼻泪管窄；干布清洁、温敷、轻按泪囊；脓多/眼红需评估 |
| 倒睫 | 鼻梁扁平所致，随发育好转，勿拔睫毛 |
| 打嗝 | 膈肌不成熟，正常反射，可自限 |
| 假月经/阴道分泌物（女） | 母体雌激素影响，自限；绿色/恶臭且持续需评估 |
| 睾丸移动/阴囊肿胀（男） | 需区分疝气与鞘膜积液，观察大小趋势，隐睾需评估 |

## 新生儿大便颜色

- 胎便（1～2 天）：黑绿色；24 小时未排需评估。
- 过渡便（3～4 天）：棕绿色。
- 金黄色（母乳）/浅黄黄绿（配方奶）：建立喂养后正常。
- 补铁时黑色属正常；**未补铁却黑便**或**灰白色便（警惕胆道闭锁）**→ 立即就医。

## 读懂哭声

- 饥饿：有节律、较低、伴吸吮动作。
- 疼痛/不适：突发尖锐、持续强烈，抱起可缓解。
- 疲倦/过度刺激：烦躁渐升级、难安抚。
- 生病：哭声弱、带鼻音，伴发热/腹泻/食欲差。
- 系统排查：尿布→互动→喂养→测温→查皮疹/异物→仍不缓解则就医。
