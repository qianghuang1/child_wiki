# 培养自主入睡

## 识别睡眠信号

- 揉眼、打哈欠、眼神发直、动作变慢、烦躁易哭。
- 出现困倦信号时**及时**让宝宝独立入睡，错过窗口反而更难入睡。

## 建立睡前程序

- 每天固定、可预测的流程（如洗澡 → 换睡衣 → 讲故事/轻柔音乐 → 关灯）。
- 保持环境安静、光线渐暗，减少兴奋刺激。
- 睡前程序有助于稳定生物钟、传递"要睡觉了"的信号。

## 避免不良入睡习惯

- 敏感识别睡眠信号，让宝宝在**清醒但困倦**时放到床上自主入睡。
- 避免养成**抱睡、摇睡、含乳头睡**等依赖性入睡习惯（否则每次夜醒都需外部帮助才能再入睡）。
