# 伤害预防

## 常见伤害与预防

| 伤害类型 | 关键预防 |
|---|---|
| 跌落 | 换尿布/高处不离手；床、沙发、桌边加防护；装窗锁、楼梯门栏 |
| 烫伤/烫伤 | 热水、热汤、热锅远离；洗澡先放冷水再兑热水并测温；避免抱孩子端热食 |
| 窒息/误吸 | 小件物品、坚果、果冻、整粒食物远离；床上无松软物；睡眠仰卧 |
| 溺水 | 水盆/浴盆/马桶随手加盖或排空；水边伸手可及看护 |
| 中毒 | 药品、日化品、清洁剂上锁存放；不将药说成"糖" |
| 电击 | 插座保护盖、电线收纳 |
| 动物咬伤 | 不让幼儿单独接触宠物/陌生动物 |

## 行为示范与安全意识

- 养育人自身遵守安全规则，做出安全示范。
- 随孩子长大，教其识别风险、建立安全行为习惯。

## 急救准备

- 掌握窒息急救和心肺复苏（见 [[急救与心肺复苏]]）。
- 记住当地急救电话与最近医院。
